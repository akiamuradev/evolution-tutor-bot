🎯 ДЛЯ КОГО ЭТОТ БОТ

═══════════════════════════════════════

ЭВО:ЛЮЦИЯ — это умный помощник ИСКЛЮЧИТЕЛЬНО для школьников 1-11 классов.

Мы НЕ работаем с бизнесом или другими аудиториями.
Мы фокусируемся на том, чтобы помочь ученикам:
• Понять сложные темы
• Подготовиться к ОГЭ и ЕГЭ
• Решать домашние задания
• Строить графики и формулы
• Создавать конспекты и документы

═══════════════════════════════════════

🎓 ПОДГОТОВКА К ОГЭ/ЕГЭ (RAG система)

═══════════════════════════════════════

Что умеет:
✅ Семантический поиск задач — ученик пишет своим языком, бот находит похожие задачи из ФИПИ
✅ Анализ трендов — прогноз тем на экзамен на основе 5+ лет данных
✅ Персональный план подготовки — адаптивное обучение под слабые места
✅ Трекинг прогресса — статистика по темам и заданиям
✅ Отчёты родителям — еженедельная статистика в Telegram

Поддерживаемые предметы:
• Математика (профиль и база)
• Русский язык
• Физика
• Химия
• Биология
• История
• Обществознание
• Информатика
• География
• Литература
• Английский язык

Источники данных:
📚 ФИПИ (официальный сайт) — демоверсии, спецификаторы, кодификаторы
📚 РешуЕГЭ/РешуОГЭ (sdamgia.ru) — 10,000+ задач с решениями
📚 Math100.ru — математика с подробными решениями
📚 Яндекс Репетитор — дополнительные задачи

Команды:
/exam — начать подготовку к экзамену
/predict [предмет] — прогноз трендов на экзамен
/practice [предмет] — случайная задача для тренировки
/solution — показать решение последней задачи
/progress — твой прогресс подготовки

💬 УНИВЕРСАЛЬНЫЙ РЕПЕТИТОР

═══════════════════════════════════════

Что умеет:
✅ Ответы на вопросы по всем предметам — математика, физика, химия, биология, история, литература и др.
✅ Подробные объяснения — 1000+ слов с примерами

🏗️ ТЕХНИЧЕСКАЯ АРХИТЕКТУРА

═══════════════════════════════════════

Модульная структура:
src/
├── bot.py — главный файл бота
├── database.py — работа с PostgreSQL
├── modules/
│   ├── ai_client.py — OpenRouter API с кэшем
│   ├── cache.py — LRU кэш (1000 записей)
│   ├── prompts.py — промпты для школьников
│   ├── keyboards.py — клавиатуры
│   └── utils.py — утилиты (конвертация символов)
├── parsers/
│   ├── universal_loader.py — загрузчик для всех предметов
│   └── sources/
│       ├── fipi_official.py — ФИПИ
│       ├── sdamgia.py — РешуЕГЭ
│       ├── math100.py — Math100
│       └── yandex_repetitor.py — Яндекс Репетитор
└── rag/
    ├── search.py — семантический поиск
    └── analyzer.py — анализ трендов

База данных:
• PostgreSQL 16 с расширением pgvector
• Векторный поиск — семантический поиск задач
• Таблицы: subjects, fipi_tasks, student_progress

ИИ модели:
• Qwen-2.5-72B — основная модель (мощная)
• Кэш — ускорение повторных запросов на 95%

• Простой вопрос — 3-5 секунд (кэш)
