# 🔒 Политика обработки персональных данных

## 1. Общие положения
1.1. Политика действует в соответствии с ФЗ №152 «О персональных данных».
1.2. Оператор: ИП [Никулина Анна Сергеевна], ИНН [345929750940], адрес: [г. Луганск Советская 20Б].
1.3. Все данные обрабатываются исключительно на серверах, расположенных на территории РФ.

## 2. Состав данных
2.1. Идентификатор Telegram (tg_id), имя пользователя, язык интерфейса.
2.2. История диалогов с ИИ (текстовые запросы и ответы) — для обеспечения работы сервиса.
2.3. Метаданные платежей (ID транзакции, статус, дата) — без реквизитов карт.

## 3. Цели обработки
3.1. Предоставление доступа к сервису и техподдержка.
3.2. Управление подпиской и учёт периодов.
3.3. Модерация контента и предотвращение нарушений.
3.4. Исполнение требований законодательства РФ.

## 4. Хранение и удаление
4.1. Данные хранятся в течение срока подписки + 3 года (требование ФНС).
4.2. История диалогов анонимизируется через 90 дней неактивности.
4.3. Пользователь может запросить удаление данных через `/support`.

## 5. Защита и передача
5.1. Данные хранятся в зашифрованной БД. Доступ только у уполномоченных лиц.
5.2. Данные не передаются третьим лицам, кроме случаев, предусмотренных законом РФ или необходимых для работы ЮKassa в обезличенном виде.
5.3. Запросы к ИИ-провайдерам передаются в обезличенном формате.

## 6. Права Пользователя
6.1. Вы можете запросить выгрузку, исправление или удаление данных через `/support`.
6.2. Отзыв согласия ведёт к прекращению доступа к сервису.
