# 🛡️ Правила использования ИИ-ассистента

## 1. Запрещённый контент
Сервис автоматически блокирует запросы, направленные на генерацию:
🔴 Материалов экстремистского или террористического характера (ФЗ №114, ФЗ №35, ст. 205 УК РФ)
🔴 Дипфейков, поддельных изображений/голоса реальных лиц без согласия (ст. 152.1 ГК РФ)
🔴 Контента, нарушающего авторские права (Netflix, Disney, Marvel и др.) без лицензии (ст. 1259, 1270 ГК РФ)
 Инструкций по изготовлению оружия, взрывчатых веществ, наркотиков
🔴 Персональных данных третьих лиц, клеветы, призывов к насилию

## 2. Модерация
2.1. Все запросы проходят автоматическую проверку перед генерацией.
2.2. При нарушении бот возвращает ошибку с указанием нормы и блокирует запрос.
2.3. Систематические нарушения ведут к приостановке подписки без возврата.

## 3. Дисклеймер
3.1. ИИ-ассистент не является источником официальной информации.
3.2. Сгенерированные материалы носят рекомендательный характер.
3.3. Пользователь самостоятельно несёт ответственность за использование результатов генерации.

## 4. Обновление правил
4.1. Исполнитель оставляет за собой право обновлять ограничения в соответствии с законодательством РФ.
4.2. Актуальная версия доступна по команде `/rules`.
